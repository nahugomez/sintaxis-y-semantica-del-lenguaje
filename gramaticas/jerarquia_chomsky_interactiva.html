Explorador interactivo de la jerarquía de Chomsky con los 4 tipos de gramáticas — corregido

3

Gramática regular

La más restrictiva — base de las expresiones regulares

Restricción

Izquierda: un solo símbolo no terminal
Derecha: a lo sumo un terminal y un no terminal

Válido

S → a
S → aT
S → Ta
S → λ

Un solo NT a la izq, máximo 2 símbolos a la der

Inválido para tipo 3

S → ab
S → aSb
S → XYb

Más de 1 terminal, o NT en el medio, o 2 NT

Probá esta gramática tipo 3

ΣT = { 0, 1 }   ΣN = { A, B }   S = A
A ::= 1B | 1    B ::= 0A
2

Gramática independiente del contexto

La de los lenguajes de programación

Restricción

Izquierda: un solo símbolo no terminal
Derecha: sin restricciones (cualquier combinación)

Válido

S → ab
S → aXb
S → bXY

Un solo NT a la izq, cualquier cosa a la der

Inválido para tipo 2

aS → a
XY → abY
aXb → aYb

Tienen más de un símbolo a la izquierda

Probá esta gramática tipo 2 — genera aⁿbⁿ

ΣT = { a, b }   ΣN = { S }   S = S
S ::= aSb | ab
1

Gramática sensible al contexto

El contexto (los vecinos) se debe mantener

Restricción

Izquierda: puede tener varios símbolos
Condición 1: el contexto (símbolos alrededor del NT) se mantiene fijo
Condición 2: |lado izquierdo| ≤ |lado derecho|

Válido

aXb → aYZb
5B6 → 53C6

Contexto fijo (naranja), solo cambia el NT (azul)

Inválido para tipo 1

Ta → aT
SXY → ab

Ta→aT: "a" cambia de lugar. SXY→ab: |3|>|2|

¿Cómo leer una regla tipo 1?
Ejemplo: 5 B 6 → 5 3C 6
Los símbolos fijos son el contexto (condición para aplicar la regla).
El símbolo que cambia es el reemplazo real (B se convierte en 3C).

Probá esta gramática tipo 1

ΣT = { a, b }   ΣN = { S, X }   S = S
S ::= abXS | ab    aX → aba    bX → bab
En aX → aba: la "a" es contexto fijo, X se reemplaza por "ba"
En bX → bab: la "b" es contexto fijo, X se reemplaza por "ab"
0

Gramática irrestricta

La menos restrictiva — todo vale

Restricción

Única regla: el lado izquierdo no puede ser vacío.
No hay restricciones de contexto, ni de longitud, ni de forma.

Válido

SXY → ab
aX → λ
aSb → ab

Todo vale: |izq| > |der|, contexto cambia, etc.

Inválido para tipo 0

λ → ab

Lado izquierdo vacío: lo único prohibido

Probá esta gramática tipo 0

ΣT = { a, b }   ΣN = { S, T }   S = S
S ::= aTb | ab    Ta → aT ← contexto NO se mantiene
Tb → bb ← |izq|=|der| pero "b" cambia de rol
G3 ⊂ G2 ⊂ G1 ⊂ G0

Cada tipo incluye al anterior. Una gramática tipo 3 también cumple las reglas del tipo 2, 1 y 0.
