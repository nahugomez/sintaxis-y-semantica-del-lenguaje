Principios clave

1. Un lenguaje puede ser generado por muchas gramáticas diferentes, incluso de distinta jerarquía.
2. Una gramática solo genera un único lenguaje.
3. El tipo del lenguaje es el más restrictivo (número más alto) entre todas las gramáticas que lo generan.
→ Este lenguaje { λ, ab, abab, ababab... } es de tipo 3, porque existe una gramática tipo 3 que lo genera.
