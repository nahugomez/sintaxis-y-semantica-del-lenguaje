Construcción interactiva de una gramática a partir del lenguaje L = {(cⁱ | b) · (ab)ʲ · aᵏ}

Lenguaje dado
L = { (ci | b) . (ab)j . ak / i > 0, j ≥ 0, k ≥ 0 }

Hay que construir la gramática que genere exactamente este lenguaje

Paso 1 — Separar por la concatenación
Sección X primera

(ci | b) con i > 0

"O bien varias c, o una sola b". Como i > 0, al menos una c.

X ::= b | T
T ::= cT | c

Genera: b, c, cc, ccc, cccc...

Sección Y segunda

(ab)j con j ≥ 0

"ab" repetido j veces. Como j ≥ 0, puede ser λ (nada).

Y ::= abY | λ

Genera: λ, ab, abab, ababab...

Sección Z tercera

ak con k ≥ 0

"a" repetida k veces. Como k ≥ 0, puede ser λ (nada).

Z ::= aZ | λ

Genera: λ, a, aa, aaa, aaaa...

Paso 2 — Gramática completa

G = ( ΣT , ΣN , S , P )

ΣT = { a, b, c }    ΣN = { S, X, T, Y, Z }    S = S
S ::= XYZ    X ::= b | T    T ::= cT | c    Y ::= abY | λ    Z ::= aZ | λ
Paso 3 — Probá la gramática

Derivá palabras para verificar que pertenecen al lenguaje

Primero expandí S, después resolvé X, Y, Z en cualquier orden
